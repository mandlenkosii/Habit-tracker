# Habit-tracker
Web app
This is web app where a user can add daily habits, mark them done, track a streak for each one, and see a summary of their progress.
## Installation
You can access the app using the provided link as it will take you to the home page of the app where you will see how the app works.
# Using the web app
You can use the app by adding one or more habits, and the habits that have been inputed will appear on the list of habits that need to done.
There is also a section where a user can marked the habits that have been done and it will update or show the number of habits that are done as well as show the progress of the habits.
